@mixin breakpoint()

aliased as bp()

Media Query Mixin

Creates a screen-only media-queries.

Since v0.0.6:

Parameters

$size: (string | number)

Enter the desired view-port width size or key from a map. If a string is used, it needs to match a map key. If a number is used, it requires a unit (px/em/rem). em is preferred.

$query: 'min-width' (string)

Will accept min-width, max-width, and print queries.

$map: $breakpoints (string)

Enter your own map to override the default map in the global settings.

@content (code block)

[CSS content for the breakpoint]

Examples

scss 
@include breakpoint(md) {
  .element {
    font-size: 2rem;
  }
}
css compiled 
@media only screen and (min-width: 61.875em) {
  .element {
    font-size: 2rem;
  }
}
scss 
.element {
  font-size: 1rem;
  @include bp(480px) {
    font-size: 1.4rem
  }
}
css compiled 
.element {
  font-size: 1rem;
}

@media only screen and (min-width: 480px) {
  .element {
    font-size: 1.4rem;
  }
}

@mixin between_breakpoints()

aliased as between_bp()

Min-Max Media Query Mixin

Creates a screen-only media-query between a min and max size.

Since v0.0.6:

Parameters

$min: (string | number)

Enter the desired view-port min-width size or key from a map. If a string is used, it needs to match a map key. If a number is used, it requires a unit (px/em/rem). em is preferred.

$max: (string | number)

Enter the desired view-port min-width size or key from a map. If a string is used, it needs to match a map key. If a number is used, it requires a unit (px/em/rem). em is preferred.

$map: $breakpoints (string)

Enter your own map to override the default map in the global settings.

@content (code block)

[CSS content for the breakpoint]

Examples

scss 
@include between_breakpoints(sm, md) {
  .element {
    font-size: 2rem;
  }
}
css compiled 
@media only screen and (min-width: 37.5em) and (max-width: 61.875em) {
  .element {
    font-size: 2rem;
  }
}
scss 
.element {
  font-size: 2rem;
  @include between_breakpoints(480px, 890px) {
    font-size: 2.4rem;
  }
}
css compiled 
.element {
  font-size: 2rem;
}

@media only screen and (min-width: 480px) and (max-width: 890px) {
  .element {
    font-size: 2.4rem;
  }
}
